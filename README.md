# StyleCast - 天気予報を、もっとあなたらしく。

**「今日の天気、どうだろう？」から「今日はどんな自分になろう？」へ。**

StyleCastは、単なる気象情報を超えて、あなたの1日を彩るためのライフスタイル提案型天気予報アプリです。
没入感のあるデザインとストーリーテリングで、毎日の天気確認をワクワクする体験に変えます。

## 🌦️ コンセプト

数字だけの天気予報に飽きていませんか？
StyleCastは、天気予報を「物語」として届けます。
「傘が必要でしょう」ではなく、「お気に入りのレインブーツで出かけたくなる雨」へ。
若者世代の感性に響くビジュアルとテキストで、その日の天気だけでなく、その日の「気分」までコーディネートします。

## ✨ 主な機能

- **ストーリーテリング天気予報**: 
  無機質なデータではなく、感情に訴えかける言葉でその日の空模様を伝えます。
- **ライフスタイル提案**:
  - **ファッション**: 天気と気温に合わせた最適なコーディネートを提案。
  - **ヘアケア**: 湿度や風に基づいたヘアアレンジのアドバイス。
  - **スキンケア**: 紫外線や乾燥レベルに合わせた肌ケアのヒント。
- **没入感のあるUI/UX**:
  天気や時間帯に合わせて変化する背景やアニメーションで、アプリを開いた瞬間からその日の世界観に浸れます。

## 🛠️ 技術スタック

このプロジェクトは以下の技術で構築されています。

- **Frontend**: [React](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: Vanilla CSS (Modern CSS Features)

## 🚀 セットアップ & 実行方法

リポジトリをクローンし、以下のコマンドで開発サーバーを立ち上げてください。

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで `http://localhost:5173` (またはコンソールに表示されるURL) にアクセスして確認してください。
